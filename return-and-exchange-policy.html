Return and Exchange Policy

Return

We accept returns only after verifying the complete opening video from unwrapping courier cover till the end without any pause and editing in the recording.

*DEFECTED AREA HAS TO BE DISPLAYED CLEARLY*

And you have to submit the opening video within 24hours of delivery. If it exceeds 24 hours from the delivery time, we won't accept any returns. So please send a request to us via email/WhatsApp along with the opening video as soon as possible.

For minor defects like stone misplacement, we don't accept returns as they can be rectified simply using jewelry sticking glue. In this case, kindly contact us, we'll guide you to fix it.

Exchange

Exchange is applicable only when the product got sold out after ordering or if you have to receive the replacement for a defective product (after verifying the opening video from the courier cover).

We don't accept the exchange if a back rope was damaged as the back rope is a complementary one. Kindly remain us in your next order we'll send you an additional back rope.

We don't accept the exchange if you have chosen the wrong bangle size while ordering (please refer to our bangle size chart before ordering). If you found any difficulty in choosing an apt size please text us and our team will assist you.

If you have any questions about this Return and Exchange Policy, please contact us.
